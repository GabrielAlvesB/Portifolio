Estou cursando Ciência da Computação na Rede de Ensino Estácio na modalidade EAD, e me especializando em programação Front End - onde utilizo muito HTML5, CSS3, JavaScript, Bootstrap e React. Adoro estar sintonizado com as novas tecnologias e desafios, no qual busco sempre solucionar da melhor forma possível.
Meus pontos fortes são a comunicação com a equipe, onde sei ouvir as dúvidas e me esforço para solucionar.Sou proativo e não me importo de passar os meus conhecimentos à colegas de trabalhos. Sou criativo e busco sempre utilizar as novas tecnologias do mercado para me auxiliar, estando em constante aprendizado. Sou apaixonado pela tecnologia e por tudo que ela pode nos oferecer.

Utilizo um sistema baseado em Ubuntu, no qual foi uma mudança muito boa, pois me fez ter mais carinho por software open source.

Principais Projetos

MediCenter

MediCenter e um projeto feito em base de um layout já pronto fornecido pelo professor Bonieky Lacerda, no curos de HTML5 & CSS3 onde tinhamos que praticar os conhecimentos adquiridos no curso, sem a utiização de frameworks apenas HTML e CSS puro, o site todo feito em Flex-box.

MediCenter

O projeto MediCenter foi solicitado pelo professor Bonieky Lacerda do curso da B7Web, como forma de aprendizagem, no qual deveríamos replicar o layout de um site utilizando HTML e CSS com flex-box.

KSB-Viagens

O projeto KSB - Viagens foi desenvolvido com o objetivo principal de aprimorar as habilidades em desenvolvimento web utilizando o framework Bootstrap. Para isso, foi utilizado um layout de referência disponibilizado no canal da DevMedia no YouTube.

KSB-Viagens

O KSB - Viagens demonstra a capacidade de desenvolver e criar um site responsivo, funcional e atraente, utilizando o framework Bootstrap como base. Além disso, esse projeto exemplifica a importância de aprender através da prática e da construção de projetos reais para aprimorar as habilidades de desenvolvimento web.

SelectCar

SelectCar é uma loja virtual de veículos que foi desenvolvida de forma que o usuário pudesse filtrar as informações do veículo desejado, como: cidade onde o veículo está, o ano e quantos quilometros o carro desejado rodou até o momento.

SelectCar

O SelectCar foi um projeto da faculdade, realizado em parceria com o Felipe Bustamante, no qual desenvolvemos uma loja virtual de veículos utilizando HTML, CSS, JavaScript e Bootstrap. As informações sobre os carros, pessoas, valores e cidades foram disponibilizados de forma randômica.
